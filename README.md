# Flask_watchlist
A watchlist web application based on Flask
## Installation  
    $ git clone https://github.com/blacwh/Flask_watchlist.git  
    $ cd watchlist
Create and activate virtual environment  

    $ python3 -m venv env  
    $ source env/bin/activate  
    $ pip install -r requirements.txt  
Generate dummy data  

    $ flask forge  
Run

    $ flask run
